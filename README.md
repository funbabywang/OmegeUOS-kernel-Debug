## 关于
此 GitHub 仓库是 Omege UOS 内核 Debug 版本，不代表最终 Releases 版本
有些代码可能有些许 Error 与 Warning

## 编译
可以直接使用 Makefile 编译，（命令为 make），也可以使用 MinGW 编译器

## Omege Kernel 版本
Omege Kernel Version Debug-1.0.0

## 内核内容
这个内核有 apt、USB Drivers、完整驱动等等

## 与 Linux kernel 的关系
这个与 Linux kernel 没有任何关系，都是自制的源代码，不要说抄袭。
